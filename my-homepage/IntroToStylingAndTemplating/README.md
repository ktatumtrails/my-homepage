# Intro to Styling and Templating in WPF

This project demonstrates the use of styles and templates in a WPF application. It showcases how to create a visually appealing user interface by applying styles to various UI elements.

## Project Structure

- **App.xaml**: Contains application-level resources and styles.
- **IntroToStylingAndTemplating.csproj**: Project file with metadata and references.
- **MainWindow.xaml**: Defines the layout and UI elements for the main window.
- **MainWindow.xaml.cs**: Code-behind for MainWindow.xaml, handling events and interactions.
- **README.md**: Documentation for the project.

## Features

- Demonstrates the use of styles for buttons and other controls.
- Shows how to create and apply templates for custom UI elements.

## Getting Started

To build and run the application:

1. Open the project in your preferred IDE.
2. Restore any necessary packages.
3. Build the solution.
4. Run the application.
