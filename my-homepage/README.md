# My Homepage

This project is a simple personal homepage that features a "Hello World" introduction, a circular profile image, and a contact button.

## Project Structure

```
my-homepage
├── index.html        # Main HTML document for the homepage
├── styles            # Directory for CSS styles
│   └── style.css     # Styles for the homepage
├── images            # Directory for images
│   └── profile.jpg   # Profile image
└── README.md         # Project documentation
```

## Getting Started

To view the homepage, open the `index.html` file in a web browser. The page includes a blue gradient background, an introduction, and a contact button.

## Customization

You can customize the homepage by editing the `index.html` and `styles/style.css` files. Replace the `profile.jpg` image with your own image by placing it in the `images` directory and updating the `src` attribute in the HTML file.
